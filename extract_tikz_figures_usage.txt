extract_tikz_figures.py — Usage Notes
======================================

PURPOSE
-------
Extracts tikz / circuitikz figures from a LaTeX source file and:
  1. Renders each figure as a high-resolution PNG in <output_dir>/
  2. Writes an editable LaTeX reference document (tikz_figures.tex) one level
     above output_dir (i.e., in the lab folder), containing all figures one per
     page with label and caption — compile with pdflatex to browse or edit.

Figures that only contain \includegraphics are skipped automatically.

REQUIREMENTS
------------
  - Python 3.10+
  - pdflatex   (MiKTeX or TeX Live — must be on the system PATH)
  - pdftoppm   (ships with MiKTeX / poppler-utils — must be on the system PATH)

USAGE
-----
Run from any directory (keep the full command on one line):

  python "<path_to_script>\extract_tikz_figures.py" <input.tex> <output_dir>

Arguments:
  input.tex    Full or relative path to the LaTeX source file.
  output_dir   The lab's media/ directory (created if absent).
               tikz_figures.tex is written to the parent of this directory.

OUTPUT PER RUN
--------------
  LabN\media\<label>-1.png   — PNG image for each figure (1200 dpi)
  LabN\tikz_figures.tex      — editable article document, one figure per page

OUTPUT NAMING
-------------
Each PNG is named after the figure's \label (with the "fig:" prefix stripped):

  \label{fig:seriesparallel}  ->  seriesparallel-1.png
  \label{fig:acpec}           ->  acpec-1.png

Figures without a \label are named fig_01-1.png, fig_02-1.png, etc.
The trailing "-1" is added by pdftoppm (always present for single-page PDFs).

EDITING FIGURES
---------------
To modify a circuit and regenerate its PNG:
  1. Open LabN\tikz_figures.tex in your LaTeX editor.
  2. Edit the TikZ / CircuitiKZ code for the relevant figure.
  3. Compile with pdflatex to preview: pdflatex tikz_figures.tex
  4. Copy the edited TikZ block back into the source LabNInstructions.tex.
  5. Re-run extract_tikz_figures.py to regenerate the individual PNG.

RESOLUTION
----------
Default: 1200 dpi.  To change, edit the '-r' value in compile_figure()
inside the script (search for 'pdftoppm').

PREAMBLE / PACKAGES
-------------------
Both the standalone (PNG) and article (tikz_figures.tex) preambles include:
  \usepackage{tikz}
  \usetikzlibrary{arrows.meta}
  \usepackage[american voltages, european currents]{circuitikz}
  \usepackage{amsmath}
  \usepackage{amssymb}
  \usepackage[table,dvipsnames]{xcolor}
  \usepackage{pgfplots}  + \pgfplotsset{compat=1.18}

Any \definecolor definitions in the source .tex are automatically carried over.
If a future lab uses additional packages, add them to STANDALONE_PREAMBLE
and ARTICLE_PREAMBLE near the top of the script.
